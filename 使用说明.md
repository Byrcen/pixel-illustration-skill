# 插画风格配图生成skill（像素风）

写完文章后，让它按 **前30% / 中间30% / 末尾30%** 自动生成 3 张像素风配图。

## 第一次使用前

打开本文件夹里的 `config.json`，把 `api_key` 换成你自己的 apimart key：

```json
{
  "api_key": "你的apimart-key"
}
```

## 怎么用

在 Claude Code 里直接说，比如：

- 「用桌面那个配图 skill 给这篇文章配图」+ 粘贴正文
- 「用桌面配图 skill，文章在 ~/Desktop/我的文章.md」
- 想改尺寸就加一句：「这次用 1:1」「要 4K」

我会读文章 → 切三段 → 各写一个像素风 prompt → 依次调 apimart 出图。

## 图存哪

`~/Desktop/文章配图/<文章名>/01_前段.png、02_中段.png、03_末段.png`

## 固定设定

- 风格：像素风 pixel art（温暖、亲和、有故事感；不写实、不赛博朋克；画面干净）
- 图内文字：一律简体中文
- 默认尺寸：16:9 横向 / 2K（可每次临时改）
- 数量：每篇 3 张

## 文件说明

- `SKILL.md` — 给 Claude 看的执行说明（核心逻辑）
- `generate.py` — 调 apimart 出图的脚本（提交→轮询→下载，纯标准库）
- `config.json` — 存你的 API key（仅本地）
